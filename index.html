XTSSH is a lite tool for SSH/SFTP

This is not commercial software. It's totally free.

I just want to make it lightweight and easy to use software:
1. only support SSH SHELL with xterm
2. only support SFTP transfer with ONE thread ONE session
3. UNICODE support
